# 项目简介

这是一个简单的实现了局域网内文件传输功能的网页应用，使用 AI 实现。  
它能帮助使用者在同一局域网下快速传输文件。只需要两台在同一局域网内的设备打开页面，然后就可以建立连接，最后愉快的发送和接收文件了。

# 环境要求

- node.js  
  需要安装 node.js
- 开放端口  
  开放 **60888** 端口和 **60999** 端口

# 运行

进入项目目录下，第一次运行时请先安装依赖

```bash
# 安装依赖
npm install
```

然后启动服务器

```bash
# 启动服务器
npm run start
```

# 使用

同一局域网下的设备通过浏览器打开 **http://localhost:60888** ，然后就可以发现其他设备了，  
选择一个设备建立连接，然后就可以愉快的传输文件了。
